Mystery Calculator

Question

There is a mystery calculator in the SETU, which can perform only two operations:

  • Double : Multiply by 2
  • Decrement : Subtract 1
Imagine that this calculator is displaying the number \(A\) on the screen at the moment. Print the minimum number of steps needed to display the number \(B\) on screen.

Sample Scenario 1
Input Format
2 3
Output Format
2

Explanation

\(2 \to 3 \to 4\)

Solution

X, Y = map(int, input().split())

def Calc(X, Y):
    steps = 0

    if X == Y:
        return steps
    else:
        if X < Y and Y % 2 == 0:
            return steps + 1 + Calc(X, Y/2)
        else:
            if X > Y:
                return steps + X - Y + Calc(X, X)
            else:
                return steps + 1 + Calc(X, Y+1)


print(int(Calc(X, Y)))

    int steps = 0;

    if (X == Y)
        return steps;
    else{
        if (X < Y && Y % 2 == 0)

        else{

            else
                return steps + 1 + calc(X, Y + 1);

    }
}
