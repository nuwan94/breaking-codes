Breaking Codes 1.0

February 2019

Question

Given a word, you need to check whether the usage of capital letters in it is valid or not.

Major Capitalization Rules to Check
  • Rule #1 - All Upper
    Word is valid if all letters in word are capitals, like "UOK".

  • Rule #2 - All Lower
    Word is valid if all letters are non capitals, like "breakingcodes".

  • Rule #3 - First Letter Upper
    Otherwise word is valid only first letter in word is capital if there is more than one letter, like "Google".
